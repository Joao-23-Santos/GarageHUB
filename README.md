🏎️ GarageHUB
A GarageHUB é uma plataforma marketplace premium dedicada à compra e venda de veículos entre utilizadores. O objetivo é simplificar a jornada de quem procura o seu próximo automóvel ou deseja vender o seu veículo atual de forma rápida, segura e intuitiva.

📱 Sobre o Projeto
O design da GarageHUB foca-se numa experiência de utilizador (UX) fluida, utilizando uma interface "Dark Mode" elegante que realça a fotografia dos veículos. A aplicação permite navegar por listagens exclusivas, filtrar por especificações técnicas e entrar em contacto direto com os vendedores.

Principais Funcionalidades:
Marketplace de Veículos: Navegação por categorias, marcas e modelos.

Filtros Avançados: Pesquisa personalizada por preço, ano, combustível e tipo de transmissão.

Perfil do Vendedor: Sistema de avaliações e histórico de listagens para garantir confiança.

Gestão de Favoritos: Guarda os teus veículos preferidos para consulta posterior.

Criação de Anúncios: Processo simplificado para publicação de veículos com upload de fotos e detalhes técnicos.

🛠️ Stack Tecnológica
Design: Figma

Frontend: Flutter

Backend e Base de dados: SupaBase
